사용법 가이드

FORLAB Plate Cropper를 사용하여 이미지에서 번호판을 크롭하는 방법을 단계별로 안내합니다.

주요 기능

AI 자동 감지

Sobel 엣지 검출과 Hough Transform 알고리즘을 결합하여 이미지에서 번호판 영역을 자동으로 찾아냅니다.

일괄 처리

여러 장의 이미지를 한 번에 업로드하고 자동으로 처리할 수 있습니다.

수동 조정

자동 감지 결과를 드래그하여 위치와 크기를 수동으로 미세 조정할 수 있습니다.

다운로드 & 공유

크롭된 이미지를 개별 또는 일괄로 다운로드하거나, 클립보드 복사 및 공유가 가능합니다.

어떻게 작동하나요?

브라우저에서 직접 실행되는 클라이언트 사이드 이미지 처리 도구입니다. 업로드된 이미지는 서버로 전송되지 않습니다.

  • Sobel 엣지 검출: 이미지의 밝기 변화를 분석하여 객체의 윤곽선을 추출합니다.
  • Hough Transform: 추출된 엣지 포인트를 기반으로 원형 패턴을 검출하여 번호판 위치를 특정합니다.
  • 적응형 크롭: 감지된 영역을 기준으로 최적의 사각형 크롭 영역을 자동 계산합니다.

기본 사용법

  • 이미지 업로드: 상단 메뉴에서 "도구" 탭을 클릭한 후, 이미지를 드래그 앤 드롭하거나 파일 선택 버튼을 클릭하여 이미지를 업로드합니다. JPG, PNG 등 주요 이미지 형식을 지원합니다.
  • 자동 감지 확인: 업로드된 이미지에서 AI가 자동으로 번호판 위치를 감지합니다. 보라색 점선 원으로 감지된 영역이 표시됩니다.
  • 위치 조정 (선택): 감지된 원의 내부를 드래그하여 위치를 이동하거나, 가장자리를 드래그하여 크기를 조정할 수 있습니다.
  • 확인 및 크롭: "확인 ✓" 버튼을 눌러 감지 결과를 승인한 후, "전체 크롭" 버튼으로 번호판을 크롭합니다.
  • 결과 저장: 크롭된 이미지는 우측 패널에 표시됩니다. 개별 저장, 복사, 공유 또는 전체 다운로드가 가능합니다.

고급 기능

  • 일괄 처리: 여러 이미지를 한 번에 업로드한 후 "전체 자동 확인" 버튼으로 모든 이미지를 한꺼번에 처리할 수 있습니다.
  • 크롭 크기 조정: 상단 슬라이더로 크롭 영역의 크기를 60%~120% 범위에서 조정할 수 있습니다.
  • 재감지: 자동 감지 결과가 정확하지 않은 경우 "재감지" 버튼으로 다시 분석할 수 있습니다.
  • 키보드 단축키: ← → 화살표 키로 이미지를 전환하고, Enter 키로 확인할 수 있습니다.
  • 추가 업로드: 작업 도중에도 "+ 사진 추가" 버튼으로 이미지를 추가할 수 있습니다.

자주 묻는 질문

Q. 이미지가 서버에 업로드되나요?

아닙니다. 모든 이미지 처리는 브라우저에서 로컬로 이루어지며, 서버로 전송되지 않습니다. 개인정보가 안전하게 보호됩니다.

Q. 어떤 이미지 형식을 지원하나요?

JPG, PNG, GIF, BMP, WebP 등 브라우저에서 지원하는 대부분의 이미지 형식을 사용할 수 있습니다.

Q. 모바일에서도 사용할 수 있나요?

네, 반응형 디자인으로 모바일 브라우저에서도 사용 가능합니다. 터치로 감지 영역을 조정할 수 있습니다.

FORLAB 소개

FORLAB은 이미지 처리 및 컴퓨터 비전 기술을 활용한 실용적인 웹 도구를 개발하는 프로젝트입니다.

프로젝트 정보

Plate Cropper는 법의학 및 보안 분야에서 번호판 이미지를 효율적으로 추출하기 위해 개발되었습니다. 기존의 수동 크롭 작업을 자동화하여 작업 시간을 크게 단축합니다.

기술 스택

  • 프론트엔드: 순수 HTML, CSS, JavaScript (프레임워크 없음)
  • 이미지 처리: Canvas API 기반 Sobel 엣지 검출, Hough Transform 원형 검출
  • 호스팅: GitHub Pages (무료 정적 사이트 호스팅)

개발 배경

보안 카메라나 현장 사진에서 번호판 영역만 빠르게 추출해야 하는 경우가 많습니다. 기존 도구들은 설치가 필요하거나 서버 업로드를 요구하여 개인정보 우려가 있었습니다. Plate Cropper는 브라우저에서 직접 실행되어 이미지가 외부로 전송되지 않으므로, 민감한 데이터도 안심하고 처리할 수 있습니다.

개인정보처리방침

최종 수정일: 2026년 3월 6일

1. 수집하는 개인정보

FORLAB Plate Cropper는 사용자의 개인정보를 직접 수집하지 않습니다. 업로드되는 이미지는 사용자의 브라우저에서만 처리되며, 서버로 전송되거나 저장되지 않습니다.

2. 쿠키 및 추적 기술

본 사이트는 Google AdSense 광고를 게재하고 있으며, Google은 광고 개인화를 위해 쿠키를 사용할 수 있습니다. Google의 쿠키 사용에 대한 자세한 내용은 Google 광고 정책을 참조하세요.

3. 제3자 광고

본 사이트는 Google AdSense를 통해 광고를 게재합니다. Google을 포함한 제3자 광고 업체는 쿠키를 사용하여 사용자의 관심사에 기반한 광고를 표시할 수 있습니다. 사용자는 Google 광고 설정에서 맞춤 광고를 비활성화할 수 있습니다.

4. 데이터 보안

모든 이미지 처리는 클라이언트 사이드에서 이루어지므로, 사용자의 이미지 데이터는 브라우저를 벗어나지 않습니다. HTTPS를 통해 안전한 연결을 제공합니다.

5. 어린이 개인정보 보호

본 서비스는 만 14세 미만의 어린이를 대상으로 하지 않으며, 의도적으로 어린이의 개인정보를 수집하지 않습니다.

6. 개인정보처리방침 변경

본 개인정보처리방침은 변경될 수 있으며, 변경 시 이 페이지에 게시됩니다.
